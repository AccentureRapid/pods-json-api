== Description ==
json API for BuddyPress. This plugin creates json api endpoints for https://github.com/WP-API

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BP json API` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Created a page for your api and assign it to the json API component.

= Manual Installation =

1. Upload `bp-json-api` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Created a page for your api and assign it to the json API component.
